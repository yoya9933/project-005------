# 成大獎學金監控來源清單

更新日期：2026-07-11

## 監控原則

- 先監控成大校內正式來源，避免校外資訊過多造成噪音。
- 每次檢查都要記錄公告日期、截止日、附件、資格條件與是否需要專題成果。
- 遇到需要「系所推薦」、「老師推薦」、「專題成果」、「研究成果」的公告，要優先標記。

## 初步分類規則

| 分類 | 說明 |
|---|---|
| A. 立即可申請 | 資格明確符合，截止日尚未過，文件可準備 |
| B. 可能可申請 | 資格需要確認，例如年級、成績、系所推薦、家庭條件 |
| C. 值得追蹤 | 目前未開放、資訊未完整，或需要等附件/辦法 |
| D. 不適合 | 身分、年級、系所、國籍、成績或截止日不符合 |

## 需要補齊

- 是否需要登入才能查看成大獎學金系統完整內容。
- KUAP 是否有網頁版公告列表可直接監控，或只能作為人工檢查來源。
- 成大水利系與工學院若有「最新消息」或「獎學金」子頁，後續應改成直接監控子頁，而不是只監控首頁。
- 成大水利系首頁目前本機自動抓取失敗，需要再找可直接讀取的公告子頁或改用瀏覽器檢查。
