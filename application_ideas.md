# 獎學金申請想法

更新日期：2026-07-11

## 使用方式

每看到一筆可能申請的獎學金，就新增一段。重點不是只抄公告，而是判斷「這筆獎學金適合用哪種專題成果包裝」。

## 通用專題成果包

先準備一份可重複改寫的成果包，之後依不同獎學金調整語氣。

### 必備材料

- 一頁式專題摘要
- 專題報告或簡報
- 成果截圖、系統畫面、模型結果、實驗結果或作品連結
- 個人貢獻說明
- 指導老師或課程證明
- 可量化成果，例如名次、分數、使用人數、改善幅度、資料規模

### 可包裝角度

| 類型 | 適合強調 |
|---|---|
| 成績型獎學金 | 學習表現、研究能力、課程成果、持續投入 |
| 清寒/助學型獎學金 | 資源有限下仍維持學習與專題投入，獎學金如何幫助完成目標 |
| 研究型獎學金 | 問題意識、方法、資料、分析、結果與延伸研究 |
| 企業型獎學金 | 實作能力、工程應用、團隊合作、產業問題解決 |
| 水利/工程領域獎學金 | 水資源、防災、永續、工程分析、公共利益與實務價值 |
| 競賽/成果型獎學金 | 作品完整度、創新性、驗證結果、展示能力 |

## 已發現機會

### 獎學金名稱：成大工學院袁福國學長獎助學金

- 來源：成大工學院
- 公告連結：https://eng.ncku.edu.tw/p/412-1014-31696.php?Lang=zh-tw
- 申請時程：https://eng.ncku.edu.tw/p/412-1014-32013.php?Lang=zh-tw
- 成果參考：https://eng.ncku.edu.tw/p/412-1014-31697.php?Lang=zh-tw
- 截止日：待從申請時程圖或最新公告確認
- 目前分類：B. 可能可申請
- 是否需要專題成果：高機率需要，尤其是國際科研競賽類
- 是否需要系所推薦：待確認
- 是否需要老師推薦：待確認

#### 適合申請的理由

工學院頁面列出的申請表包含「出席國際學術會議」與「參加國際科研競賽」兩種用途，成果頁也列出多個學生團隊與競賽案例，包含水利所獲補助紀錄。若你的專題成果能連到水利、防災、永續、水資源、工程應用或競賽展示，這筆值得優先追蹤。

#### 可使用的專題成果

待填：需要你提供目前已有的專題題目、報告、簡報、程式、模型、實驗結果、競賽或發表紀錄。

#### 建議包裝角度

把專題寫成「工程問題解決」而不是只有課堂作業。重點放在問題背景、方法、可驗證成果、你的個人貢獻，以及參加會議或競賽能如何讓成果被檢驗與延伸。

#### 需要補的材料

- 最新申請時程與申請資格
- 申請表
- 專題摘要
- 成果證明
- 指導老師或系所推薦需求
- 預計參加的會議或競賽資訊

#### 申請風險

如果沒有實際會議、競賽或可對應的科研活動，可能不適合立即申請。需要先確認是否只補助已確定參加的活動。

#### 下一步

確認你的專題成果是否能對應「國際科研競賽」或「出席國際學術會議」。若目前還沒有明確活動，可先把這筆列為追蹤，等符合活動出現時快速套用成果包。
