# 獎學金申請想法

更新日期：2026-07-11

## 使用方式

每看到一筆可能申請的獎學金，就新增一段。重點不是只抄公告，而是判斷「這筆獎學金適合用哪種專題成果包裝」。

## 通用專題成果包

先準備一份可重複改寫的成果包，之後依不同獎學金調整語氣。

### 必備材料

- 一頁式專題摘要
- 專題報告或簡報
- 成果截圖、系統畫面、模型結果、實驗結果或作品連結
- 個人貢獻說明
- 指導老師或課程證明
- 可量化成果，例如名次、分數、使用人數、改善幅度、資料規模

### 可包裝角度

| 類型 | 適合強調 |
|---|---|
| 成績型獎學金 | 學習表現、研究能力、課程成果、持續投入 |
| 清寒/助學型獎學金 | 資源有限下仍維持學習與專題投入，獎學金如何幫助完成目標 |
| 研究型獎學金 | 問題意識、方法、資料、分析、結果與延伸研究 |
| 企業型獎學金 | 實作能力、工程應用、團隊合作、產業問題解決 |
| 水利/工程領域獎學金 | 水資源、防災、永續、工程分析、公共利益與實務價值 |
| 競賽/成果型獎學金 | 作品完整度、創新性、驗證結果、展示能力 |
